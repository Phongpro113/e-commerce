## 🧱 PROMPT 1 — KHỞI TẠO PROJECT TỪ ZERO (QUAN TRỌNG NHẤT)
Bạn là một WordPress + WooCommerce expert.

Tôi có một server trống (folder rỗng). Hãy hướng dẫn và tạo toàn bộ cấu trúc website e-commerce bằng WordPress + WooCommerce từ đầu.

YÊU CẦU:
- Giải thích từng bước thật cụ thể để tôi làm theo
- Không bỏ qua bước nào
- Dùng WooCommerce làm core

STACK:
- WordPress
- WooCommerce

OUTPUT:
1. Checklist cài đặt từ folder trống
2. Cách setup database + hosting
3. Cách cài WordPress thủ công (nếu cần)
4. Cài WooCommerce và cấu hình ban đầu

Quan trọng:
- Viết theo dạng từng bước (Step 1, Step 2...)
- Tôi sẽ làm theo từng bước và báo bạn trước khi sang bước tiếp theo

## ⚙️ PROMPT 2 — SETUP SHOP CHUẨN BÁN HÀNG

Tiếp tục từ website WooCommerce đã cài xong.

Hãy cấu hình website thành một shop bán hàng hoàn chỉnh:

YÊU CẦU:
- Cấu hình tiền tệ VND
- Setup thanh toán: COD + VNPay + MoMo (nếu cần plugin)
- Setup shipping đơn giản (flat rate + GHN nếu có thể)
- Tạo trang cần thiết:
  - Home
  - Shop
  - Cart
  - Checkout
  - My account

OUTPUT:
- Hướng dẫn từng click trong admin WordPress
- Plugin cần cài (chỉ những plugin cần thiết nhất)

## 🎨 PROMPT 3 — GIAO DIỆN BÁN HÀNG
Thiết kế giao diện WooCommerce tối ưu bán hàng.

YÊU CẦU:
- Chọn theme nhẹ, dễ dùng (ví dụ Astra hoặc Flatsome)

- Tối ưu tốc độ load

OUTPUT:
- Theme nên dùng
- Cách cài và setup
- Cấu trúc homepage chuẩn chuyển đổi cao

## 📦 PROMPT 4 — THÊM SẢN PHẨM CHUẨN BÁN ĐƯỢC
Hướng dẫn tôi setup sản phẩm trong WooCommerce để bán hàng hiệu quả.

YÊU CẦU:

OUTPUT:
